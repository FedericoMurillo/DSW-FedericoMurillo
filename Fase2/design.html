Proposito del Sitio:

Sitio web para aprender a usar las etiquetas basicas HTML, atributos y valores afianzando el conocimiento de las personas que esten interesadas en aprender el lenguaje de etiquetas HTML

Etiquetas basicas de HTML

El conjunto de las etiquetas basicas constituyen un documento completo de HTML 5, donde principalmente se encuentran estas etiquetas:
!DOCTYPE HTML, que es la declaracion inicial, e indica a los navegadores de que se trata el documento HTML 5.
despues van las etiquetas de apertura y cierre que son html /html, dentro de estas etiquetas ira todo el documento
las etiquetas head /head para la cabecera, dentro de esta va informacion sobre la pagina como el titulo, la codificacion de caracteres, etiquetas para los buscadores y mas,
luego van las etiquetas de cuerpo body /body, dentro de ellas ira todo el contenido que el navegador mostrara cuando se abra el documento.
