Etiquetas basica de HTML

Html es un markup lenguage, lo que significa que esta escrito con
codigos que puede leer una persona sin que sea necesario compilarlos primero
Cuando se escribe un codigo en HTML, estas escribiendo etiquetas HTML
Todas las etiquetas HTML estan hechas con un numero de partes especificas, incluyendo

  • -El caracter "menos que" <
  • -Una palabra o caracter que determina que etiquetas se esta escribiendo
  • -Cualuier numero de atributos HTML que se quiere usar, escritos de la forma nombre="valor"
  • -El caracter "mayor que" >

    • Hay unas serie de etiquetas que son las mas usadas para crear cualquier documento
    de HTML:

    -body: para el contenido
    -head: para informacion sobre el docuemtno
    -div: divison dentro del contenido
    -a: para enlaces
    -strong: para poner el texto en negrita
    -br para saltos de linea
    -H1 - H6 para titulos dentro del contenido
    -img: para añadir imagenes al documento
    -ol: para listas, ul: para listas desordenadas, li: para elementos dentro de la lista
    -p: para parrafos
    -span: para estilos de una parte del texto
