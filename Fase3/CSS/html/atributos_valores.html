Atributos y Valores de HTML

Son las propiedades que se pueden asignar a los elementos
a HTML, se colocan siempre en la etiqueta de apertura, esto para añadir
complementar informacion o significado al elemnto HTML
al que estamos conectando. Se debe mencionar que:

-Los atributos deben estar en la etiqueta inicial, nuenca en la final
-El orden de los atributos dentro de la etiqueta es indiferente
-Deben separarse con un espacion en blanco del nombre del elemento y de otros atributos
-Atributo y valor deben estar unidos por el signo: =
-Los valores deben estar encerrador con comillas, dobles o siples.
